Welcome to CASL's VERAView application.


System Requirements
-------------------

VERAView expects the Canopy Python environment.  Refer to 'canopy-install.pdf'
for a guide to installing and configuring Canopy for VERAView.


Windows Users
-------------

You must launch using the 'veraview.run.bat' script file.  You cannot
the Python file 'veraview.py' directly.


Mac Users
---------

Similarly, you must launch using the 'veraview-mac.run.sh' script file.
You cannot the Python file 'veraview.py' directly.


Linux Users
-----------

That's right, you must launch using the 'veraview-linux.run.sh'
script file.  You cannot the Python file 'veraview.py' directly.
