Mac OS X Python Installation and Configuration

Options

If you are running a more recent Mac OS X version, you probably already have Python-2.7. In this case, using pip to install the needed modules is the best way to go. Otherwise, you're best bet is to use MacPorts or Homebrew to install/build Python-2.7. I have only used the former on a 10.6.8 system and will reference it below.

Already Have Python-2.7

Install with Pip

If you don't have Pip, grab the source from here and run this with your Python-2.7 executable: 

$ python get-pip.py

This should install a script named pip in your Python directory. It can be used to install (download source and build) any modules you're missing except for wxPython. It will take care of downloading dependent modules as well. You can test to see if you need a module by trying to import it in a Python script or interactively. The modules you need are:

  • h5py
  • matplotlib
  • numpy (installed by h5py)
  • Pillow (PIL)
  • pyparsing (installed by matplotlib)
  • python-dateutil (installed by matplotlib)
  • six (installed by matplotlib)

You install a module with a simple command, such as: 

$ pip install h5py

Install wxPython

A pre-built wxPython is available here. Install it as you would any other Mac package. You'll have to have elevated privileges to do so.

Using MacPorts

If you don't have it, get it pre-built for later versions of Mac OS X or get the source. Go here. You can get the source here.

Note I tried an install in my home direcotry, but libraries for required modules just wouldn't build. Do the default install to /opt/local worked but requires elevated privileges.

After building and/or installing MacPorts do a self update and install Python-2.7 and the required modules, including wxPython. 

# port selfupdate
# port install python27

The list of packages you'll need to install with port install package-name is:

  • py27-h5py
  • py27-matplotlib
  • py27-Pillow
  • py27-wxpython-3.0
  • py27-numpy (installed by py27-h5py)
  • py27-parsing (installed by py27-matplotlib)
  • py27-dateutil (installed by py27-matplotlib)
  • py27-six (installed by py27-matplotlib)
